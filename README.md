Plugin to support Thrift language in IntelliJ

How to build
===============

[JDK 11 or later](https://adoptium.net/) is required to build from source.

```bash
./gradlew :thrift:buildPlugin
```

How to run locally with new changes
===============

```bash
./gradlew :thrift:runIde
```
